NetBeans缓存更新

更新NetBeans后，ide的提示经常出错，通过更新ide缓存即可解决该问题。更新方式吗，那就是暴力删除了:)，如果怕删除太危险，可以移到备份。
NetBeans的缓存地址如下：

mac     /Users/<username>/Library/Caches/NetBeans/xxx/
windows C:\Users\<username>\AppData\Local\NetBeans\Cache\xxx\
unix    /home/<username>/.cache/netbeans/xxx/
